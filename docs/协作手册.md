## 3 模块接口规范

### 3.1 BlockDevice（A 提供 → 所有人使用）

```cpp
// include/core/block_device.h
namespace pfs {

class BlockDevice {
public:
    // 创建虚拟磁盘（内存分配）
    // total_blocks: 总块数，block_size: 每块字节数（默认 512）
    BlockDevice(uint32_t total_blocks, uint32_t block_size = 512);
    ~BlockDevice();

    // 读一个块到 buf，buf 必须 >= block_size
    int read_block(uint32_t block_no, void* buf);

    // 把 buf 写入一个块
    int write_block(uint32_t block_no, const void* buf);

    // 整盘持久化到宿主机文件
    int save_to_file(const char* path);

    // 从宿主机文件加载
    int load_from_file(const char* path);

    // 查询
    uint32_t total_blocks() const;
    uint32_t block_size() const;

    // 清零整个磁盘
    void zero_all();
};

}  // namespace pfs
```

**调用约定：**
- 返回 0 成功，-1 失败
- block_no 从 0 开始
- 线程不安全（单线程项目）

### 3.2 IFileSystem — VFS 接口（A 定义 → A 实现 UnixFs，B 实现 Fat16Fs）

```cpp
// include/core/vfs.h
namespace pfs {

struct DirEntry {
    char name[14];       // 文件名
    uint16_t inode_no;   // UNIX: inode 号，FAT16: 首簇号
    uint8_t type;        // 0=file, 1=dir
    uint32_t size;       // 文件大小（字节）
};

struct FileStat {
    uint16_t mode;       // 权限
    uint16_t uid;        // 所有者
    uint16_t gid;        // 组
    uint32_t size;       // 大小
    uint32_t atime;      // 访问时间
    uint32_t mtime;      // 修改时间
    uint32_t ctime;      // 创建时间
    uint16_t nlink;      // 硬链接数
    uint8_t type;        // 0=file, 1=dir
};

struct DiskUsage {
    uint32_t total_blocks;
    uint32_t used_blocks;
    uint32_t total_inodes;   // FAT16 填 0
    uint32_t used_inodes;    // FAT16 填 0
};

class IFileSystem {
public:
    virtual ~IFileSystem() = default;

    // --- 文件系统生命周期 ---
    virtual int  fs_format() = 0;                    // 格式化
    virtual int  fs_mount() = 0;                     // 挂载（从磁盘加载元数据到内存）
    virtual int  fs_unmount() = 0;                   // 卸载（写回元数据）

    // --- 文件操作 ---
    virtual int     fs_create(const char* path, uint16_t mode) = 0;
    virtual int     fs_open(const char* path, int flags) = 0;  // 返回 fd
    virtual int     fs_close(int fd) = 0;
    virtual ssize_t fs_read(int fd, void* buf, size_t len) = 0;
    virtual ssize_t fs_write(int fd, const void* buf, size_t len) = 0;
    virtual int     fs_delete(const char* path) = 0;

    // --- 目录操作 ---
    virtual int  fs_mkdir(const char* path) = 0;
    virtual int  fs_rmdir(const char* path) = 0;
    virtual int  fs_chdir(const char* path) = 0;
    virtual int  fs_ls(const char* path, std::vector<DirEntry>& out) = 0;
    virtual std::string fs_pwd() = 0;

    // --- 元信息 ---
    virtual int  fs_stat(const char* path, FileStat& out) = 0;
    virtual int  fs_chmod(const char* path, uint16_t mode) = 0;
    virtual int  fs_link(const char* src, const char* dst) = 0;

    // --- 系统信息 ---
    virtual std::string fs_type_name() const = 0;    // "UNIX" 或 "FAT16"
    virtual DiskUsage   fs_disk_usage() const = 0;
};

}  // namespace pfs
```

**调用约定：**
- 成功返回 0（或 fd/字节数），失败返回 -1
- path 支持绝对路径 `/home/alice/file.txt` 和相对路径 `subdir/file.txt`
- flags: `O_READ=1, O_WRITE=2, O_APPEND=4`
- mode: 八进制权限如 `0755`

namespace pfs {

class OpenFileTable {
public:

### 3.4 UserManager（B 实现 → Shell 使用）

```cpp
// include/core/user_manager.h
namespace pfs {

class UserManager {
public:
    // 登录，成功返回 user_id，失败返回 -1
    int login(const char* username, const char* password);

    // 注销
    int logout(uint16_t user_id);

    // 创建用户（仅 root）
    int add_user(const char* username, const char* password,
                 uint16_t uid, uint16_t gid);

    // 修改密码
    int change_password(uint16_t user_id, const char* old_pw, const char* new_pw);

    // 权限检查：user_id 对 mode 为 file_mode 的文件是否有 access 权限
    bool check_access(uint16_t user_id, uint16_t file_uid,
                      uint16_t file_gid, uint16_t file_mode,
                      uint8_t access);  // READ=1, WRITE=2, EXEC=4

    // 查询
    uint16_t current_uid() const;
    uint16_t current_gid() const;
    std::string current_username() const;
};

}  // namespace pfs
```

### 3.5 Shell 命令注册（B 实现）

```cpp
// include/shell/command_registry.h
namespace pfs {

// 命令处理函数签名
// fs: 当前文件系统引擎，args: 命令参数（不含命令名）
// output: 输出字符串（TUI 会显示它）
using CmdHandler = std::function<int(IFileSystem& fs,
                                     UserManager& um,
                                     const std::vector<std::string>& args,
                                     std::string& output)>;

class CommandRegistry {
public:
    void register_cmd(const std::string& name, CmdHandler handler,
                      const std::string& usage);

    // 解析并执行一条命令行
    int execute(const std::string& cmdline, IFileSystem& fs,
                UserManager& um, std::string& output);

    // 列出所有命令及用法
    std::vector<std::pair<std::string, std::string>> list_commands();
};

}  // namespace pfs
```

## 4 集成规范

### 4.1 依赖方向（严格单向）

```
TUI → Shell → VFS(IFileSystem) → FS Engines → BlockDevice
                  ↑
             UserManager
```

- **禁止反向依赖**：core 不能 include shell 或 tui 的头文件
- **禁止跨层调用**：TUI 不能直接调用 BlockDevice
- Shell 通过 `IFileSystem&` 引用调用 FS，不关心是 UNIX 还是 FAT16

### 4.2 命名空间

所有代码在 `namespace pfs {}` 内，避免与系统头文件冲突。

### 4.3 错误处理

- 返回 int：0 成功，-1 失败
- 错误信息通过 `output` 字符串参数传递给上层
- 不使用异常（C++ exception）

### filesysorig 已知 bug（报告可用）

1. `namei()` 返回 NULL(0) 与合法索引 0 冲突
2. `bfree()` 缺少要释放的块号参数
3. `iput()` 中误用 `balloc()` 应为 `bfree()`

5. `_dir()` 内外层循环变量 `i` 冲突
6. `open.c` 中存在判断逻辑取反
7. `login()` 中 `strcmp` 返回值判断错误
